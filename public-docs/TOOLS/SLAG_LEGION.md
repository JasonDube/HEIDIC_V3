# SLAG LEGION

**SLAG LEGION** is a game built with HEIDIC and EDEN Engine. It serves as both a demonstration of HEIDIC's capabilities and a full-featured game project.

## Overview

SLAG LEGION showcases HEIDIC's game development features, including:
- Entity Component System (ECS) with query iteration
- Hot-reload for rapid iteration
- Vulkan rendering via EDEN Engine
- Resource management
- Game-specific systems and mechanics
